Create `src/articles/history.py` with these exact contents:

**Imports:** `re` from stdlib, `datetime` from `datetime`, `Path` from `pathlib`

**Module constant:**
```python
HISTORY_DIR = Path(__file__).parent / "history"
```
This resolves to `src/articles/history/` in dev (pip install -e .) per user decision.

**Public functions:**

1. `save_article(url: str, title: str, content: str) -> Path`:
   - Call `HISTORY_DIR.mkdir(exist_ok=True)`
   - Iterate `HISTORY_DIR.glob("*.md")` — for each file, call `_read_url_from_frontmatter(path)`. If it matches `url`, call `path.unlink()` and `break` (duplicate handling — overwrites old file)
   - Generate date prefix: `datetime.now().strftime("%y%m%d")`
   - Generate safe title: `_sanitize_title(title)[:50]` — if result is empty string, use `"untitled"`
   - Build filename: `f"{date_prefix}-{safe_title}.md"`
   - Build filepath: `HISTORY_DIR / filename`
   - Write file: `filepath.write_text(f"---\nurl: {url}\n---\n\n{content}")`
   - Return `filepath`

2. `list_history() -> list[dict]`:
   - If `not HISTORY_DIR.exists()`: return `[]`
   - Initialize empty `entries` list
   - For each path in `sorted(HISTORY_DIR.glob("*.md"), reverse=True)`:
     - `title = _title_from_filename(path.stem)`
     - `url = _read_url_from_frontmatter(path)`
     - Append `{"title": title, "url": url, "path": path}`
   - Return `entries`

3. `load_article_content(path: Path) -> str`:
   - `text = path.read_text()`
   - If `text.startswith("---\n")`: find `end = text.find("\n---\n", 3)`. If `end != -1`: return `text[end + 5:]` (content after closing frontmatter marker)
   - Otherwise return `text`

**Private helpers:**

4. `_sanitize_title(title: str) -> str`:
   - `clean = re.sub(r"[^\w\s-]", "", title)` — strip everything except word chars, spaces, hyphens
   - `clean = re.sub(r"\s+", "-", clean.strip())` — collapse whitespace to single hyphens
   - Return `clean`

5. `_title_from_filename(stem: str) -> str`:
   - If `len(stem) > 7 and stem[6] == "-" and stem[:6].isdigit()`: set `title_part = stem[7:]`
   - Else: `title_part = stem`
   - Return `title_part.replace("-", " ")`

6. `_read_url_from_frontmatter(path: Path) -> str`:
   - Wrap in try/except `(OSError, UnicodeDecodeError)` returning `""`
   - `text = path.read_text()`
   - If `text.startswith("---\n")`: find `end = text.find("\n---\n", 3)`. If `end != -1`: extract `frontmatter = text[4:end]`
   - Iterate `frontmatter.split("\n")`: if a line `startswith("url: ")`, return `line[5:].strip()`
   - Return `""` if no URL found
